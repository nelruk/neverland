## Objetos R
Hay cinco tipos de objetos en R:
  * character()
  * numeric()
  * integer()
  * complex()
  * logical(TRUE, FALSE)
El operador **:** se utiliza para crear intersecuencia, por ejemplo
x <- 1:6
print (x)

## Tipos de vectores
x <- c(TRUE, FALSE) ## logical
x <- c(0.5,0.6) ## numerical
x <- c("a", "b", "c") ## character
x <- 9:29 ## integer
x <- (1+0i, 4+2i) ## complex

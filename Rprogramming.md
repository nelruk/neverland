## Objetos R
Hay cinco tipos de objetos en R:
  * character()
  * numeric()
  * integer()
  * complex()
  * logical(TRUE, FALSE)
El operador **:** se utiliza para crear intersecuencia, por ejemplo
```  
     x <- 1:6
      print (x)
      [1] 1 2 3 4 5 6
````
## Tipos de vectores
````
    logical = x <- c(TRUE, FALSE)
    numerical = x <- c(0.5,0.6)
    character = x <- c("a", "b", "c")
    integer = x <- 9:29
    complex = x <- (1+0i, 4+2i)
```
